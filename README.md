# FCTV Desktop

Netflix-style IPTV player. Supports M3U playlists, DASH/HLS, ClearKey DRM, custom headers.

## Build via GitHub Actions
1. New GitHub repo → upload all files → commit
2. Actions tab → wait ~5 min
3. Download **FCTV-Windows-Installer** artifact → run .exe

## Shortcuts
- Space = Play/Pause  |  F = Fullscreen  |  Esc = Close player
- Ctrl+F = Search  |  ↑↓ = Volume
